# 新规则

- 可合成南瓜头（craftableCarvedPumpkin）

  使用剪刀和南瓜合成南瓜头，每合成一个南瓜头剪刀-1耐久（受耐久附魔影响）。



# 网络

- 现在玩家进入服务器时会在服务器后台比对发送客户端与服务端的Carpet AMS Addition的版本信息。



# 修复

- 修复[#195](https://github.com/Minecraft-AMS/Carpet-AMS-Addition/issues/195)。
- 修复使用`新的磁石配方（useNewLodestoneRecipe）`规则没有正确识别为配方规则的问题。



# 其他

- 支持Minecraft 1.21.10。



---



# New Rules

- craftableCarvedPumpkin

  Using shears and a pumpkin to craft a carved pumpkin, each crafted carved pumpkin reduces the shears durability by 1 (affected by unbreaking enchantments).




# Network

- Now, when players join the server, the server backend compares the version information of the Carpet AMS Addition mod between the client and the server.



# Fix

- Fix [#195](https://github.com/Minecraft-AMS/Carpet-AMS-Addition/issues/195)。
- Fixed an issue where the `useNewLodestoneRecipe` rule was not correctly recognized as a recipe rule.



# Other

- Support Minecraft 1.21.10.

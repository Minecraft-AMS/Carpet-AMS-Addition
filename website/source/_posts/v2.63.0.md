# 更改

**下界传送门不传送玩家(playerNoNetherPortalTeleport)** 规则升级为以下规则：

- **下界传送门不传送玩家（commandPlayerNoNetherPortalTeleport）**
  玩家无法通过下界传送门进行传送。
  开启规则后，使用`/plerNoNetherPortalTeleport help`命令查看使用说明

# Changes

# 新规则

- **AMS网络协议（amsNetworkProtocol）**
  AMS网络协议的开关。

- **获取客户端玩家FPS（commandGetClientPlayerFps）**
  使用指令获取指定客户端玩家的FPS。
  开启规则后，使用“/getClientPlayerFps help”命令查看使用说明

- **设置玩家姿势（commandSetPlayerPose）**
  使用指令设置玩家姿势。
  开启规则后，使用“/playerPose help”命令查看使用说明

- **AMSP网络协议Debug指令（commandAmspDebug）**
  
  一些用于网络通信测试的指令。

# New Rules

* **amsNetworkProtocol**
  The switch of the AMS network protocol.

* **commandGetClientPlayerFps**
  Use command to get the FPS of the specified player.

* **commandSetPlayerPose**
  Use command to set player pose.

* **commandAmspDebug**
  
  Some commands for network communication testing.
